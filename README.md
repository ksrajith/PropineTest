# PropineTest

##### Overview
--------------------------
You can test the 'https://vast-dawn-73245.herokuapp.com/' UI through this repo.

##### Installation Instructions
###### Prerequisites:

* Java 1.8

* Maven > 3.3.9

Compile with Maven:

run from project root: `mvn clean compile`

Run tests:

run `mvn test`
